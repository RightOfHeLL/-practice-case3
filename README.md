1. HTML (index.html): Создан каркас, включающий контейнер слайдера, информационный блок (счетчик текущего изображения), область для вывода изображения (<img>) и блок управления с кнопками «Назад» и «Вперед». 
2. Стилизация (style.css): Для создания привлекательного интерфейса использовалась технология Flexbox для центрирования элементов. Контейнеру добавлены скругления углов (border-radius), тень (box-shadow) для эффекта объема. Изображениям задано свойство object-fit: cover для корректного масштабирования без искажений. Кнопкам добавлены эффекты при наведении (hover) и нажатии (active).
3. Логика работы (script.js): Функционал реализован через массив строк с URL-адресами изображений. Созданы слушатели событий (addEventListener) для кнопок. Реализована логика зацикливания: при достижении конца массива и нажатии кнопки «Вперед», индекс сбрасывается на 0 (первое изображение), а при нажатии «Назад» на первом элементе — индекс переключается на последний элемент массива. При каждом клике динамически обновляется не только атрибут src у картинки, но и текстовое содержимое счетчика (например, "Изображение 2 из 5").
